simple-hot-toast

Toasts made simple. Pure JS. React-style. No fluff.
A lightweight, zero-dependency toast notification library inspired by React Hot Toast.
Library developed by Vinay Chaudhary.

🚀 Installation

Include this in your HTML:

<script src="toast.js"></script>

📦 Basic Usage

toast.success("Success!");
toast.error("Something went wrong.");
toast.loading("Please wait...", { autoCompleteAfter: 3000 });

🎯 API Reference

  • toast.success(msg, options?)
  • toast.error(msg, options?)
  • toast.loading(msg, options?)
  • toast.custom(html, options?)
  • toast.promise(promise, { loading, success, error }, options?)

📍 Positions

"top-left" | "top-center" | "top-right"
"bottom-left" | "bottom-center" | "bottom-right"

💡 Custom HTML Toast

toast.custom("<strong style='color: green;'>✅ Done!</strong>", {
  position: "bottom-center"
});

🔄 Async Handling with toast.promise

toast.promise(fetch(url), {
  loading: "Loading...",
  success: "Done!",
  error: "Failed!"
});
